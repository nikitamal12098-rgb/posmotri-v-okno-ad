# Проект "Посмотри в окно"

## Описание прокта
Данный сайт показывает города России утром, днём, либо ночью из окна многоэтажек.  
Сайт состоит из блока видео, занимающего большую часть, а также  
из возможности выбора определённого города. На данный момент на  
сайте представлены __только часть городов__, но в будущем  
планируется расширить список. Каждый может зайти на данный сайт,  
и, быть может, найдёт свой город с видом из окна
## Инструкция по использованию
1. Выбираете в правой чатсе сайта город (присутствует скролл-бар)
2. Выбираете время суток (выбор представлен под видео)
3. Нажимаете кнопку **"Найти"** и ждёте
   * Если видео найдётся, то вы его увидете
   * Если видео нет, то покажется ошибка
4. Наслаждаемся видами из окна своего города
